Angular Starter Template

Checklist:
1) Install Node and WebStorm
2) Terminal:
   cd \src
   npm start

** 'npm run bump' will bump the patch version up a number.
This should install: Express, Browserify, LiveReload, Gulp, Angular, AngularStrap, AngularUI Router, etc.
