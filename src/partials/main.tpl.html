Setup for Brand NEW Machines

  1. Shared Workspace (& Notifications):
    • Create a Slack account
    • Slack Workspace: https://proridium.slack.com/
    • Install the appropriate client for build notifications (Mac [iTunes or Direct], Chrome, Android & iOS)

  2. Server: We use NodeJS.

    • Download and install it. Please make sure it is in the path.
    • Confirm that it is installed property. 
      node --version

  3. Local Source Control: We use a git

    • Download Git and install it.
    • Confirm that it is installed property. 
      git --version
    • Follow the setup instructions.
  4. Source Control Host: We use GitHub to host our files in a private repository.
    • Create a GitHub account if you do not already have one.
    • Use Two-Factor authentication
  5. Tying Local & Remote Source Control:
    • Setup your local git installation for use with GitHub
      • including authentication
    • Optional: Install a Git Client on your desktop
      • GitHub for Mac
      • GitHub for Windows
      • Tower2 for Mac *I use this one.
      • SourceTree for Windows or Mac
  6. Get Permissions:
    • Send your GitHub & slack account login names to me: shartzog@gmail.com so I can add you to the team.
  7. Prepare the Code for Dev: Start with our default project template and add it to your github account.
    • Wait for my email saying you have access to our source on GitHub
    • Open the template repo on github
    • Fork it
  8. Dev Environment: Install WebStorm

Project Setup (Terminal):

  1. npm init: This will install all of the node package requirements, then install/download the client dependencies with bower.
  2. npm start: This should install: Express, Browserify, LiveReload, Gulp, Angular, AngularStrap, AngularUI Router, etc. This will then run the gulp init command for you. ** npm run bump: This will bump the patch version up a number.

WebStorm Setup:

  1. Configure a Node.js App (Guide to Node.js Development with WebStorm):
    • Create a node Configuration
  2. Get the Code:
    • Configure WebStorm for use with GitHub.
    • Enable the GitHub bundled plugin to get access to GitHub integration.
    • Register your GitHub account in WebStorm.
    • Clone your GitHub forked repo of the web repo.
